# ⚽ UEFA Champions League 2021-22 EDA

An Exploratory Data Analysis (EDA) project on the UEFA Champions League 2021-22 season using Python. The project analyzes player and team performance through data cleaning, statistical analysis, and visualizations.

---

## 📌 Project Overview

This project explores the UEFA Champions League 2021-22 dataset to identify key performance trends among players and teams. Python was used for data cleaning, analysis, and visualization to generate meaningful football insights.
